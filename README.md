## About

This library provides Material Design Icons as TypeScript exports.
All icons are generated automatically from the official Material Design Icons.

This project exists thanks to these projects:
- [Material Design Icons](https://pictogrammers.com/library/mdi/)
- [svg-to-ts](https://github.com/nivekcode/svg-to-ts)

## Installation

With npm:

> npm install @nationalbankbelgium/mdi-ts

With yarn:

> yarn add @nationalbankbelgium/mdi-ts
